# 🛒 Customer Basket Analysis using MySQL

## 📌 Project Overview

This project analyzes customer basket behavior using MySQL. The objective is to explore customer demographics, basket activity, and purchasing patterns using SQL queries. The project demonstrates SQL concepts ranging from basic data exploration to advanced analytical queries.

---

## 🎯 Objectives

* Analyze customer demographics.
* Study customer basket activity.
* Perform JOIN operations between multiple tables.
* Identify top customers based on basket count.
* Rank customers using SQL Window Functions.
* Generate business insights from customer purchasing behavior.

---

## 🛠️ Technologies Used

* MySQL 8.0
* MySQL Workbench
* SQL
* GitHub

---

## 📂 Dataset

The project uses two datasets:

### 1. customer_details.csv

* customer_id
* sex
* customer_age
* tenure

### 2. basket_details.csv

* customer_id
* product_id
* basket_date
* basket_count

2. Create the database.
3. Import both CSV files.
4. Execute the SQL queries in `project_queries.sql`.
5. Review the query outputs and screenshots.
