# Business Insights

## Project Objective

Analyze customer basket behavior using SQL by combining customer demographics with basket purchase information.

## Key Insights

* Imported and analyzed customer and basket datasets using MySQL.
* Performed JOIN operations to combine customer and basket information.
* Analyzed customer distribution based on gender.
* Calculated average customer age and average basket size.
* Identified the Top 10 customers based on basket count.
* Ranked customers using SQL Window Functions.
* Identified customers with above-average basket activity using subqueries.

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)
* INNER JOIN
* Subqueries
* Window Functions (RANK)
* LIMIT
* DESCRIBE
* SHOW TABLES

## Conclusion

This project demonstrates practical SQL skills for customer behavior analysis and showcases database querying, data exploration, joins, aggregations, subqueries, and ranking techniques commonly used in data analytics.
